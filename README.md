<h1 align="center">Tutora</h1>
<h3 align="center">AI-powered tutoring platform</h3>

<p align="center">
  Tutora is an engineering thesis project focused on building a modern, scalable and secure platform for online tutoring.
  The system integrates real-time communication, an interactive whiteboard, shared learning materials, and AI-generated notes and tests.
  The architecture is designed as a distributed microservices ecosystem with a Next.js web client.
</p>
